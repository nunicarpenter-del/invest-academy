# Feature Spec: Document Vault & VOD Academy

## 1. Secure Document Vault
* **Two-Way Interaction:** Both Clients and Analysts/Admins can upload, view, and download files.
* **Storage Security:** Strict Row Level Security (RLS) policies on the storage buckets. A client can NEVER access a file uploaded to another client's directory.

## 2. VOD Center (The Academy)
* **Content Delivery:** A categorized video library for ongoing financial education.
* **Retention Tool:** Videos are added continuously to keep alumni clients returning to the app.
