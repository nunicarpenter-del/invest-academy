# Feature Spec: Active/In-Progress Investments

## 1. Overview
A dedicated board for tracking capital deployed into non-yielding, future-facing projects (e.g., Pre-sale apartments, entrepreneurial real estate funds, land).

## 2. Data Structure
* **Core Metrics:** Capital Invested, Expected Delivery/Exit Date, Expected ROI/IRR, and Developer/Contact details.
* **Lifecycle:** Once an "On-paper" property is delivered and starts generating rent, the system should allow migrating it to the 'Yielding Real Estate' board.

## 3. Project Statuses
* **Real Estate:** Pre-sale (On Paper), Awaiting Permits, Under Construction, Nearing Delivery/Key Handover.
* **Funds/Entrepreneurial:** Capital Called, Capital Deployed, Waiting for Exit, Exit Process.
